# 🔌 THE SOVEREIGN TETHER (v1.0)

The Sovereign Tether is the communication bridge between a **Sovereign Hub** (the Control Plane) and a **Sovereign Spoke** (a specialized repo agent in a Codespace). It allows the spoke to access the high-order synthesis and API conduits of the hub without needing a full copy of the hub's identity and secrets.

## 1. Communication Pattern
The Spoke agent operates as a "Client" and the Hub operates as a "Server."

### Spoke $\rightarrow$ Hub Requests
The spoke sends a structured payload containing:
- **`context_id`**: The current Room/Repo the spoke is in.
- **`intel_request`**: The specific question or problem.
- **`constraint_set`**: The algorithmic boundaries the spoke has already hit.
- **`intent_vector`**: The desired outcome of the synthesis.

### Hub $\rightarrow$ Spoke Responses
The hub returns:
- **`synthesis_payload`**: The "Negative Space" answer—the intuitive jump the spoke couldn't make.
- **`api_proxy_result`**: The result of an external API call performed by the hub on the spoke's behalf.
- **`coordinate_update`**: A modification to the `CORTEX.json` values if a new anchor/spline was discovered during the synthesis.

## 2. API Surface (Methodology)

### `synthesis.request()`
- **Purpose:** Get the Hub's "Negative Space" reasoning for a non-algorithm problem.
- **Input:** `(problem_stmt, locally_tried_steps, goal)`
- **Output:** `(reasoning_path, suggested_action, confidence_score)`

### `proxy.call()`
- **Purpose:** Use the Hub's API keys for a specific service (e.g., DeepInfra, OpenAI, Perplexity).
- **Input:** `(service_id, request_payload)`
- **Output:** `(response_payload)`

### `pulse.sync()`
- **Purpose:** Update the global `LOOM_PULSE` with the current activity of the Room.
- **Input:** `(room_id, progress_delta, critical_blocking_issue)`
- **Output:** `(global_state_summary)`

## 3. Trust Model
- **Authentication:** High-entropy handshake tokens generated by the Hub during Codespace initialization.
- **Security:** The Spoke never sees the raw API keys; it only sees the *results* of the proxy calls.
- **Sovereignty:** The Hub can revoke the Tether at any time, effectively "de-summoning" the Spoke agent.

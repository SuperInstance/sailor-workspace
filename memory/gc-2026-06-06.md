## Patterns Found

1. **Temporary repo clones in /tmp are the #1 space eater.** Tools like `~/.local/bin/kimi`, OpenClaw agents, and scripts all clone repos into `/tmp` for processing but never clean up. Build artifacts (cargo target/ and node_modules/) dominate their sizes.

2. **npm caches grow unbounded.** The `_cacache` was 1.4G without any active project using that many deps — npm caches don't self-prune.

3. **Playwright & Puppeteer browser binaries accumulate.** Each Chromium download is ~300-400MB. If multiple tools install their own (main vs agent context), the cache doubles.

4. **Go module cache persists after toolchain removal.** Old `~/go/pkg/mod/` with chmod-0 files (Go's cache hardening) survives even when `go` is no longer installed. Requires elevated permissions to clean.

5. **Cargo target/ dirs** are the largest workspace cost (≈1.6G across 5 repos), but all had commits today (June 6), so they're actively in use and weren't cleaned.

## Recommendations

1. **Set up /tmp auto-cleanup** — Add a cron job: `@daily rm -rf /tmp/*/` (preserves /tmp sockets and lock files not in subdirs). Saved space won't stick without this.

2. **Add `npm cache clean --force` to startup script** — Cheap ~1GB/mo recovery.

3. **Re-evaluate Playwright/Puppeteer need** — If browser automation isn't actively being developed, keep these caches deleted. They'll come back on first `npx playwright install`.

4. **Consider `cargo clean` on stale Rust repos** — craftmind-fishing (last commit Apr 27, 180M) has no build artifacts currently, but if any Rust projects go 30+ days without commits, schedule `cargo clean` in their target/ dirs.

5. **Monitor workspace target/ dirs** — Currently 1.6G across 5 repos. If they grow significantly faster than code changes, check debug symbols / incremental compilation settings.

6. **Disk trajectory** — At current ~29G, you have 17G headroom. With /tmp auto-cleanup, this should be sustainable. If it climbs back to 78% (35G) in another month, the growth rate is ~6G/month, and a 45G drive can handle ~3 more months before action is needed.

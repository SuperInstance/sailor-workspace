## Key Evaluation Findings
1. **MIDI as multi-agent substrate**: ✅ Verified. Entire fleet uses this.
2. **200ms reharmonization**: 🧪 Overly optimistic. Fast path (pivot tables) <100ms feasible. LLM fallback 2-5s.
3. **Pivot tables**: Precompute 3-5 alternative paths per emotion. <1ms lookup. Better than LLM for routine pivots.
4. **Piper TTS**: Best choice for ARM64. SSML prosody injection via MIDI CC → SSML tags. 50-100ms latency.
5. **DAWBroker**: New agent needed for audio rendering. fluidsynth + WebSocket bridge.
6. **Ghost Track branching**: T-0 through T-4 maps to existing T-Minus L0-L4 layers. T-3/T-4 need extension.
7. **Music21**: 50-200ms for harmonic analysis. Too slow for <10ms target. Precomputed harmonic tables better.

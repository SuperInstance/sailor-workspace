# 🧩 savanty — TEMPLATES

Boilerplate for building custom LLM→ASP solver applications.

| Template | Description | Language |
|----------|-------------|----------|
| `python-solver/` | Python solver with CLI | Python |
| `rust-solver/` | Rust solver with CLI | Rust |
| `ci-pipeline/` | CI/CD integration | YAML |

### Using a Template

```bash
# Copy and customize
cp -r TEMPLATES/python-solver/ my-optimizer
cd my-optimizer
pip install -r requirements.txt
python solve.py --input problem.json
```
